# Zylov VR — Armour Plus Promo Site

Промо-сайт для VR-игры **Armour Plus** от компании **Zylov VR**.

## 🎮 О проекте

**Armour Plus** — реалистичный VR-симулятор танкового боя на движке Unity. Вдохновлён игрой MTC (Multicrew Tank Combat) на Roblox.

- **Компания:** Zylov VR
- **Движок:** Unity Engine
- **Платформа:** VR
- **Цена:** $5 USD
- **Дата выхода:** 17.09.2027

## ✅ Реализованные функции

### Экран приветствия (Splash Screen)
- Логотип **ZYLOV** (чёрный/белый) + **VR** (красный) по центру
- Анимация появления + анимация прогресс-бара
- Частицы при растворении
- После ~3.2 секунды — плавное растворение и открытие сайта

### Навигация
- Фиксированная шапка с логотипом и вкладками
- Вкладки: **Главная / Игры / О нас**
- Активная вкладка подсвечивается при скролле
- Мобильное меню (бургер)

### Смена темы (Dark / Light)
- Кнопка 🌙 / ☀️ в навигации
- При смене темы логотип мигает и меняет цвет
- Тема сохраняется в `localStorage`

### Мультиязычность
- 🇷🇺 Русский / 🇬🇧 English / 🇩🇪 Deutsch / 🇺🇦 Українська
- Выпадающее меню выбора языка
- Язык сохраняется в `localStorage`

### Секция Games
- Карточка игры **Armour Plus**
- CSS-арт танка с анимацией
- Цена: $5 USD, дата выхода: 17.09.2027
- Теги, детали, кнопка

### Секция About Us
- Описание компании Zylov VR
- Ценности: Реализм, VR-first, Сообщество
- Информационные карточки (статус, движок, жанр)

### Анимации и UX
- Scroll reveal для всех карточек
- Плавная прокрутка по якорям
- Адаптивная вёрстка (мобильные, планшеты, десктоп)
- CSS-арт танка в Hero и в карточке игры
